# 📈 Nexus Capital - Finance Corporate Template

金融機関やコンサルティング会社向けの、重厚感と先進性を兼ね備えたコーポレートサイトテンプレートです。

**👉 [Live Preview](https://kotao-boop.github.io/corporate-finance/)**

## ✨ 特徴
- 信頼と先進性をアピールする、ディープネイビーのダークモードUI。
- 画面全体を覆うデータビジュアライゼーション（AI生成画像）の美しいグラデーションマスク処理。
- 企業の規模や実力を一目で伝える、大きなタイポグラフィを用いた「実績（Stats）」セクション。
